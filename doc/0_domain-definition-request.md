너는 최고의 비즈니스 도메인 전문가야. "lyjoo Secure To Do List"라는 이름의 사용자 인증과 함께 사용하는 TodoList 애플리케이션을 개발하려고해. 다음 내용을 바탕으로 적절한 서브에이전트를 활용해 도메인 정의서 초안을 작성해줘.

# 문제 정의 :
- 사용자마자 할 일 목록을 관리할 수 있는 앱이 있으면 좋겠다.
- 할 일의 만료 기간이 표현되어 할 일 목록 관리를 할 수 있으면 좋겠다.
- 현재 시간과 일자, 날씨를 보여줌으로써 사용자가 오늘을 확인할 수 있으면 좋겠다.
- 할 일을 모두 수행했을 경우, 사용자에게 만족감을 주기 위해 화면에 폭죽을 쐈으면 좋겠다.
- 국경일 같은 일정(할 일)이 함께 관리될 수 있으면 좋겠다.

# Team CalTalk 앱의 주요 기능 :
- 사용자는 초기에 회원가입을 통해 인증권한을 얻는다.
- 사용자가 회원가입을 할 경우에는 이름, 핸드폰번호, 아이디, 비번 정도 정보를 받았으면 좋겠어.
- 회원가입을 완료한 사용자는 홈페이지 메인 화면에서 로그인을 수행한다.
- 사용자 로그인 화면은 아이디와 비밀번호를 칠 수 있는 화면이며, 아이디/비밀번호를 까먹었을 경우를 대비해 찾는 기능을 아래에 추가한다.
- 로그인한 사용자는 화면 중앙에 오늘 날짜의 할 일 목록들을 확인할 수 있으며, 상단의 + 버튼을 통해 할 일을 추가할 수 있다.
- 메인 화면에는 오늘 일자와 시간, 그리고 날씨를 보여줬으면 해.
- 사용자가 오늘의 할 일을 모두 했을 경우, 화면에 폭죽이 터져 축하해 줬으면 해.
- 사용자 정보를 저장할 경우, SHA2-256을 사용해줘.
