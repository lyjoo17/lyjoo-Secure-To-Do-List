## 🏗️ WHS-TodoList 와이어프레임 (v1.0)

### 1. 공통 레이아웃 (Main Layout)
모든 로그인 후 페이지에 적용되는 기본 구조입니다.
- **헤더**: 로고, 사용자 정보, 다크모드 토글
- **사이드바**: 메뉴 이동 (데스크톱 기준, 모바일은 하단 탭 또는 햄버거 메뉴)

```text
+---------------------------------------------------------------+
| [로고] lyjoo-secure-to-do-list  [🌙] [사용자님 ▼] [로그아웃]    |  <-- Header
+---------------------------------------------------------------+
|           |                                                   |
|  [메뉴]   |                                                   |
|           |                                                   |
| 📋 할일   |            (페이지별 컨텐츠 영역 - Outlet)         |
| 🗑️ 휴지통 |                                                   |
| 📅 국경일 |                                                   |
| ⚙️ 설정   |                                                   |
|           |                                                   |
|           |                                                   |
+---------------------------------------------------------------+
```

### 2. 로그인 페이지 (Login Page)
사용자의 첫 진입 화면입니다.

```text
+---------------------------------------------------------------+
|                                                               |
|              [ lyjoo-secure-to-do-list 로고 ]                 |

|        |  이메일 주소                                |        |
|        +---------------------------------------------+        |
|                                                               |
|        +---------------------------------------------+        |
|        |  비밀번호                                   |        |
|        +---------------------------------------------+        |
|                                                               |
|        [           로그인 (Primary Color)            ]        |
|                                                               |
|             계정이 없으신가요?  [회원가입 하기]               |
|                                                               |
+---------------------------------------------------------------+
```

### 3. 할일 목록 페이지 (Main / Todo List)
**핵심 기능**이 모여 있는 페이지입니다. (PRD 5.1.2 참조)
- 상단에 **오늘의 국경일** 배너 (PRD 5.1.4 참조)
- 필터링 탭 및 추가 버튼

```text
+---------------------------------------------------------------+
|  📢 [오늘의 국경일] 🇰🇷 광복절 - 대한민국 독립 기념일          | <-- Holiday Banner
+---------------------------------------------------------------+
|                                                               |
|  2025년 11월 26일 (수) ⛅ 24°C (OpenWeather)                  |
|                                                               |
|  [ 전체(5) ] [ 진행중(3) ] [ 완료(2) ]      [ + 할일 추가 ]   | <-- Filter & Add Btn
|                                                               |
|  -----------------------------------------------------------  |
|  [ ] ⬜ PRD 문서 작성 완료하기                    [수정][삭제]|
|      🕒 11.25 ~ 11.26  [D-day] [높음]                         |
|  -----------------------------------------------------------  |
|  [ ] ⬜ DB 스키마 설계하기                        [수정][삭제]|
|      🕒 11.26 ~ 11.27  [D-1]                                  |
|  -----------------------------------------------------------  |
|  [v] ☑️ (완료) 프론트엔드 초기 세팅               [수정][삭제]| <-- 완료된 상태(취소선)
|      🕒 11.25 ~ 11.25                                         |
|  -----------------------------------------------------------  |
|                                                               |
|  * 오늘의 모든 할일 완료 시 화면 전체에 폭죽 효과(Confetti) 🎉  |
+---------------------------------------------------------------+
```

### 4. 할일 추가/수정 모달 (Todo Modal)
`Todo` 엔티티의 필드를 입력받는 팝업입니다. (ERD 참조)

```text
+---------------------------------------------------------------+
|  할일 추가                                          [ X ]     |
|---------------------------------------------------------------|
|                                                               |
|  제목 (필수) *                                                |
|  +-------------------------------------------------------+    |
|  | 예: 프로젝트 기획서 완성하기                          |    |
|  +-------------------------------------------------------+    |
|                                                               |
|  내용 (선택)                                                  |
|  +-------------------------------------------------------+    |
|  | 상세 내용을 입력하세요...                             |    |
|  |                                                       |    |
|  +-------------------------------------------------------+    |
|                                                               |
|  기간 설정                                                    |
|  [ 2025-11-26 ] (시작일)  ~  [ 2025-11-28 ] (종료일)      |
|                                                               |
|---------------------------------------------------------------|
|                  [ 취소 ]    [ 저장하기 ]                     |
+---------------------------------------------------------------+
```

### 5. 휴지통 페이지 (Trash Page)
삭제된 할일을 관리합니다. (PRD 5.1.3 참조)
- 삭제 대신 **복원**과 **영구 삭제** 버튼이 존재합니다.

```text
+---------------------------------------------------------------+
|  🗑️ 휴지통 (최근 30일 보관)                   [ 휴지통 비우기 ] |
|---------------------------------------------------------------|
|                                                               |
|  -----------------------------------------------------------  |
|  ❌ 잘못 만든 할일                                            |
|     삭제일: 2025.11.25                     [ ↩️ 복원 ] [ 🔥삭제 ] |
|  -----------------------------------------------------------  |
|  ❌ 테스트용 할일                                             |
|     삭제일: 2025.11.24                     [ ↩️ 복원 ] [ 🔥삭제 ] |
|  -----------------------------------------------------------  |
|                                                               |
|  * 휴지통의 항목은 복원하거나 영구 삭제할 수 있습니다.        |
|                                                               |
+---------------------------------------------------------------+
```

### 🎨 디자인 스타일 가이드 요약 (PRD 참조)
이 와이어프레임을 실제 UI로 구현할 때 적용할 스타일입니다.

1.  **메인 컬러**: `Primary Green (#00C73C)` - 버튼, 강조 텍스트, 체크박스 활성 상태
2.  **배경색**: `White (#FFFFFF)` / `Light Gray (#F3F4F6)` (리스트 배경)
3.  **폰트**: 시스템 폰트 (San-serif), 깔끔하고 가독성 높게
4.  **반응형**:
    *   **Desktop**: 사이드바 노출, 2단/3단 레이아웃
    *   **Mobile**: 사이드바 숨김(햄버거 메뉴), 리스트 카드 형태 간소화
